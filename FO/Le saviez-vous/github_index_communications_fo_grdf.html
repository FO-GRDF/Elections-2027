✅ Ce que tu peux modifier en priorité
Étape 1
Remplacer les liens
Dans chaque carte, change l'attribut href pour pointer vers le bon fichier HTML publié sur ton dépôt.
Étape 2
Adapter les titres
Renomme chaque carte avec l'intitulé exact de la communication pour que la page soit immédiatement compréhensible.
Étape 3
Ajouter ou supprimer des cartes
Tu peux dupliquer un bloc carte pour ajouter une nouvelle communication, ou en retirer si tu veux une version plus courte.
Étape 4
Publier sur GitHub Pages
Une fois les fichiers en place, cette page devient ta vitrine centrale pour accéder à tout le reste.
